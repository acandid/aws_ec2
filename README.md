# Ansible Role: aws_ec2
=========

A simple Ansible role for create EC2 instance in AWS


Requirements
------------
Before starting you need the following packages on your ansible server
- epel-release
- python2-pip
- boto
- boto3


Role Variables
--------------


None of the variables below are required
